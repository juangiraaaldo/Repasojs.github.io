# 📌 CRUD Datos - JavaScript Moderno

Proyecto académico desarrollado con **JavaScript, Bootstrap 5 y
localStorage**, aplicando conceptos modernos como:

-   CRUD con almacenamiento local
-   Callbacks
-   Promises
-   Async/Await
-   Destructuring Objects

------------------------------------------------------------------------

## 🚀 Funcionalidades

✔ Registrar usuarios (nombre, documento y correo)\
✔ Editar registros\
✔ Eliminar registros\
✔ Validar que correo y documento sean únicos\
✔ Guardar información en localStorage\
✔ Exportar datos en formato JSON

Además incluye ejercicios prácticos de JavaScript moderno.

------------------------------------------------------------------------

## 🛠 Tecnologías Utilizadas

-   HTML5
-   CSS3
-   Bootstrap 5
-   JavaScript (ES6+)
-   localStorage

## 📚 Actividades Implementadas

### 1️⃣ CRUD con localStorage

-   Crear registros
-   Editar registros
-   Eliminar registros
-   Validación de datos únicos

### 2️⃣ Callbacks

Búsqueda en dos bases de datos usando funciones callback.

### 3️⃣ Promises

Promesa que verifica si una palabra termina en vocal.

### 4️⃣ Async/Await

Función asíncrona que retorna el doble de un número después de 2
segundos.

### 5️⃣ Destructuring Objects

Extracción de propiedades de un objeto usando destructuring.
